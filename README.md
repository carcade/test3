## Требования
- NodeJS >= 10
- Yarn 1.22.4

## Установка
Установить зависимости
```shell script
yarn install
```

## Использование
Запуск встроенного веб-сервера
```shell script
yarn serve
```

Запуск линтера
```shell script
yarn lint
```

Запуск development сборки
```shell script
yarn build
```

Запуск production сборки
```shell script
yarn prod
```

Запуск сборки с отслеживанием изменений
```shell script
yarn watch
```
